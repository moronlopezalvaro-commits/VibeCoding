IA utilizada: Gemini Pro

# Prompt 1
Quiero crear una Landing Page interactiva que consista en una Tier List de videojuegos de la saga "Inazuma Eleven" utilizando solo HTML, CSS y JavaScript (no quiero que utilices ningún framework). El código debe estar separado en index.html, style.css y script.js. La web debe funcionar y verse bien en escritorio, tablets y móviles.

Debe de haber filas clasificadas por rangos: S (Divino), A (Legendario), B (Bueno), C (Normal), D (Malo). Cada fila debe tener un color distintivo a la izquierda (S en verde oscuro, A en verde, B en amarillo, C en naranja y D en rojo) y un área amplia a la derecha para soltar las imágenes. Debajo de las filas haz una zona llamada "Banquillo" donde aparezcan inicialmente todas las carátulas de los juegos. Implementa la lógica para poder arrastrar las imágenes desde el "Banquillo" a cualquier fila de la Tier List, y también moverlas entre filas.

Inspírate en la estética del anime Inazuma Eleven: usa una paleta de colores con azules eléctricos, amarillos rayo y naranjas. El fondo de la web debe ser oscuro o tener un degradado sutil que recuerde a un campo de fútbol o al espacio (estilo Galaxy). Usa fuentes modernas y dinámicas para los títulos. Cuando se pase el ratón por encima de una carátula, que esta aumente ligeramente de tamaño.

Para las imágenes utiliza las que te doy, en mi proyecto están guardadas en una carpeta llamada “juegos”. Haz que todas tengan la misma proporción y resolución.
